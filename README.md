# ai4scholar Skill Hub

论文科研写作 Skills 集合，适用于 **Workbuddy**（腾讯）和 **OpenClaw**。

## 使用方式

**Workbuddy**

进入技能管理 → 上传技能，将对应文件夹中的 `SKILL.md` 拖入上传即可。

**OpenClaw**

```bash
openclaw skills install ai4scholar/<skill-name>
```
